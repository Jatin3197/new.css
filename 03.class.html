Introduction of CSS

CSS stands for Cascading style sheet.In Css is used multiple fonts,styles,formatting,colors,background color, gradient,text properties like textalign,textshadow,text decoration,text indent etc.CSS is used for to make a beautiful webpages and websites.CSS is used for decoration and designing purpose.

CSS selectors

CSS selectors is used to select any thing like paragraph,heading(h1 to h6),class,id and many more tags like header,footer, section, nav etc.It is perform some styling in the style tag using the selectors like h2, paragraph, class,id etc.

CSS Styling

CSS styling is used in multiple way like inline, internal ,external.For styling it is used style tag Inline stlying is performed inside body tags like heading,pargraph etc.Internal styling are performed in head tag. External styling is performed in other file to be linked with this file.

CSS elements

CSS are some elements like font,color,background,text etc.In font there are many things like font size,font family,font style,font-weight,etc. In color there are many things to pick a color like color picker,using RGB to pick a color,using hex to pick a color,using hsl to pick a color. In background it is also used many things like background-color,bg image, etc.In text it is used many things like text align,text decoration,text indent,text shadow etc.

Color picker elements

-There are three types of color picker elements

  • RGB
    • 255,0,0:red
    • 0,255,0:green
    • 0,0,255:blue
  • HEX
    • #ff0000 is red
    • #00ff00 is green
    • #0000ff is blue
  • HSL
    • 0 to 360 degree is red
    • 120 is green
    • 240 is blue
